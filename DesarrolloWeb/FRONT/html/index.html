h1,h2,...,h6-Headings

Las etiquetas h1 a h6 se utilizan para definir encabezados HTML. H1 define el encabezado más importante. H6 define el encabezado menos importante. Nota: Solo use un h1 por página, esto debe representar el encabezado / asunto principal para toda la página. Además, no omita los niveles de encabezado: comience con h1, luego use h2, y así sucesivamente.

Por ejemplo:

Este es el heading 1

Este es el heading 2

Este es el heading 3

Este es el heading 4

Este es el heading 5
Este es el heading 6

div-Division

La etiqueta div define una división o una sección en un documento HTML. La etiqueta div se utiliza como contenedor para elementos HTML, que luego se diseñan con CSS o se manipulan con JavaScript. La etiqueta div se diseña fácilmente utilizando el atributo class o id. ¡Cualquier tipo de contenido se puede poner dentro de la etiqueta div! Nota: De forma predeterminada, los navegadores siempre colocan un salto de línea antes y después del elemento div.

Por ejemplo:

El elemento div

Este es un encabezado en un elemento div

Este es texto en un elemento div.

Este es texto afuera del elemento div.

span-Color

La etiqueta span es un contenedor en línea que se utiliza para marcar una parte de un texto o una parte de un documento. La etiqueta span es fácilmente diseñada por CSS o manipulada con JavaScript usando el atributo class o id. La etiqueta span es muy parecida al elemento div, pero div es un elemento de nivel de bloque y span es un elemento en línea.

Por ejemplo:

Mi madre tiene ojos verdes.

strong-TextImportance

La etiqueta strong se utiliza para definir texto con gran importancia. El contenido interior se muestra normalmente en negrita. Consejo: ¡Usa la etiqueta b para especificar texto en negrita sin ninguna importancia adicional!

Por ejemplo:

Este texto es importante!

i-Italic

La etiqueta i define una parte del texto en una voz o estado de ánimo alternativo. El contenido interior se muestra normalmente en cursiva. La etiqueta i se usa a menudo para indicar un término técnico, una frase de otro idioma, un pensamiento, el nombre de un barco, etc.

Por ejemplo:

Lorem ipsum es el mas popular texto de relleno en la historia.

El RMS Titanic, unn lujoso barco de vapor, se hundio el 15 de Abril de 1912 despues de golpear un iceberg.

u-Misspelled/Underline

La etiqueta u representa un texto que no está articulado y tiene un estilo diferente al texto normal, como palabras mal escritas o nombres propios en texto chino. El contenido interior se muestra normalmente con un subrayado. Puede cambiar esto con CSS (ver ejemplo a continuación). Consejo: ¡Evite usar el elemento u donde podría confundirse con un hipervínculo!

Por ejemplo:

Esto esta mal escrito.

em-Emphasized

La etiqueta em se utiliza para definir el texto enfatizado. El contenido interior se muestra normalmente en cursiva. Un lector de pantalla pronunciará las palabras en em con énfasis, utilizando el estrés verbal.

Por ejemplo:

Nosotros no podemos vivir asi.

img-Imagen

La etiqueta img se utiliza para incrustar una imagen en una página HTML.

Las imágenes no se insertan técnicamente en una página web; Las imágenes están vinculadas a páginas web. La etiqueta img crea un espacio de retención para la imagen a la que se hace referencia.

La etiqueta img tiene dos atributos obligatorios:

src: especifica la ruta de acceso a la imagen
alt: especifica un texto alternativo para la imagen, si la imagen por algún motivo no se puede mostrar

Nota: Además, especifique siempre el ancho y el alto de una imagen. Si no se especifica ancho y alto, la página puede parpadear mientras se carga la imagen.

Consejo: Para vincular una imagen a otro documento, simplemente anide la etiqueta img dentro de una etiqueta a (ver ejemplo a continuación).

Los principales idiomas de la Web

HTML es el lenguaje de marcado estándar para crear páginas Web. HTML describe la estructura de una página Web, y consta de una serie de elementos. Los elementos HTML indican al navegador cómo mostrar el contenido.

CSS es un lenguaje que describe cómo se deben mostrar los elementos HTML en pantalla, papel o en otros medios. CSS ahorra mucho trabajo, ya que puede controlar el diseño de varias páginas web a la vez.

JavaScript es el lenguaje de programación de HTML y la Web. JavaScript puede cambiar el contenido HTML y los valores de atributos. JavaScript puede cambiar CSS. JavaScript puede ocultar y mostrar elementos HTML, y más.

ul-Bullet

La etiqueta ul define una lista desordenada (con viñetas). Utilice la etiqueta ul junto con la etiqueta li para crear listas desordenadas. Consejo: Usa CSS para aplicar estilos a las listas. Consejo: Para listas ordenadas, usa la etiqueta ol.

Por ejemplo:

  • Café
  • Té
  • Leche

ol-Ordered list

La etiqueta ol define una lista ordenada. Una lista ordenada puede ser numérica o alfabética. La etiqueta li se utiliza para definir cada elemento de la lista. Consejo: Usa CSS para aplicar estilos a las listas. Consejo: Para la lista no ordenada, utilice la etiqueta ul.

Por ejemplo:

  1. Coffee
  2. Tea
  3. Milk

  2. Tea
  3. Milk
